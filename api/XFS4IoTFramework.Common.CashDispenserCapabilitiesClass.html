    ShutterControl

    If set to TRUE the shutter is controlled implicitly by the Service. If set to FALSE the shutter must be controlled explicitly by the application using the Dispenser.OpenShutter and the Dispenser.CloseShutter commands. This property is always true if the device has no shutter. This field applies to all shutters and all positions.
