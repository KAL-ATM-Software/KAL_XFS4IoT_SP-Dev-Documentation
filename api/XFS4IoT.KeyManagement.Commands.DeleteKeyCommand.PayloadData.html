    Authentication

    This property can be used to include authentication data if required by the command.

    Additionally, if the command requires authentication:

    • The KeyManagement.StartAuthenticate command must be called before this command to.
    • Commands which do not clear or modify the authentication data from the device may be executed between the KeyManagement.StartAuthenticate and keyManagement.Authenticate command requests.
    • If prior to this command request, KeyManagement.StartAuthenticate is not called or a command clears the authentication data from the device, sequenceError will be returned.
