    SetAutostartupTime(SetAutostartupTimeRequest, CancellationToken)

    This command is used to set the time at which the machine will automatically start. It is also used to disable automatic start-up.If a new start-up time is set by this command it will replace any previously set start-up time. Before the auto start-up can take place the operating system must be shut down. This will only be called if the device reports AutoStartupMode in the capabilities.
