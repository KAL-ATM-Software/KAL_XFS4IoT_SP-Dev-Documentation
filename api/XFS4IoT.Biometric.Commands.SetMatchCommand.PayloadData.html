    CompareMode

    Specifies the type of match operation that is being done. The following values are possible:

    • verify\t- The biometric data will be compared as a one-to-one verification operation.
    • identity - The biometric data will be compared as a one-to-many identification operation.

    Identifier

    In the case where compareMode is verify this parameter corresponds to a template that has been imported by a previous call to the Biometric.Import. If compareMode is identify a comparison is performed against all of the imported templates, in which case this property can be null. This property corresponds to the list of template identifiers returned by the Biometric.GetStorageInfo command. id1
