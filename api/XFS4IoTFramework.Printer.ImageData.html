    Data

    Image data The bit data format is: (Device-Independent Bitmaps without header) every scanline is 4 bytes aligned and the scanlines are stored upside down, with the first scan in memory being the bottommost scan in the image. if the device needs to have format first scanline to be topmost, need to re-order bit format to suits your printer device.
