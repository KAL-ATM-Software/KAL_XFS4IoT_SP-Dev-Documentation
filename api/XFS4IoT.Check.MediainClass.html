    LastMediaOnStacker

    Contains the number of media items on the stacker successfully accepted by this instance of the command execution. May be null if it is unknown or the device does not have a stacker.

    The number of refused media items can be determined by lastMedia - lastMediaOnStacker. This is only possible if these values contain known values, and would not be possible if a bunch of items were refused as a single entity. 3
